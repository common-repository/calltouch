Плагин для интеграции с сервисом [Calltouch](https://www.calltouch.ru/).

== Description ==
Интеграция с Calltouch позволит вам эффективно анализировать отдачу ваших рекламных кампаний:
вы узнаете сколько тратите на рекламу, сколько эта реклама приносит вам звонков и заявок, и главное,
сколько эти лиды приносят вам реальной выручки.

Плагин встраивает js-скрипт [Calltouch](https://www.calltouch.ru/) на все страницы сайта.
Для подключения нужен идентификатор счетчика сайта в личном кабинете Calltouch, если личный кабинет еще не
зарегистрирован, то для регистрации необходимо вначале [отправить заявку](https://www.calltouch.ru/why-calltouch/?utm_source=wordpress&utm_medium=cms&utm_campaign=wordpress_app&utm_content=script_app) на подключение.
[Политика в отношении обработки персональных данных](https://www.calltouch.ru/privacy_policy.pdf)

== Installation ==
1. Вам необходимо авторизоваться в личном кабинете Calltouch, выбрать сайт, на который необходимо установить скрипт.
2. Выбрав сайт, необходимо кликнуть на пункт меню «Интеграции» и выбрать «API и Webhooks».
3. На странице настройки вам необходимо передвинуть ползунок вправо, включив «Передавать данные по API». После этого, вам необходимо скопировать значение из поля «ID счетчика (mod_id)».
4. Скопированное значение необходимо вставить в поле «ID счетчика Calltouch» на странице настроек в «Подключение к Calltouch» и нажать кнопку «Сохранить».
